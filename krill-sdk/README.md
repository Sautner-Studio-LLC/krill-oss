# Krill SDK for Java and Kotlin Multiplatform

The Krill SDK is a Kotlin Multiplatform library that provides a consistent API for interacting with the Krill platform across all supported platforms (Raspberry Pi, Windows, Linux, macOS, Android, iOS). It abstracts away the underlying implementation details and allows you to build custom integrations, automations, and applications on top of Krill.

Learn more here: [krillswarm.com](https://krillswarm.com/categories/open-source)
